语法

ここ／そこ／あそこ／こちら／そちら／あちら

名词 は 场所あです。

表示场所、人或物等存在的地方。

どこ／どちら

名词 の 名词

こ／そ／あ／ど　一览表

こ系列 そ系列 あ系列 ど系列
物 これ それ あれ どれ
物、人 この名词 その名词 あの名词 どの名词
场所 ここ そこ あそこ どこ
方向、场所（郑重） こちら そちら あちら どちら

お〜

放在与说话人或第三者有关的事物之前，表达说话人的敬意。

「お」国はどちらですか。
