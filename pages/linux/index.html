Linux常用命令

# 查看端口占用

lsof(list open files)，在linux中，一切皆文件。

lsof -i tcp:8080

lsof -i :8080

  • lsof -i:端口号

  • netstat -tunpl | grep 端口号

    • -t tcp
    • -u udp
    • -n 拒绝显示别名，能显示成数字的全转化成数字
    • -l listen状态的
    • -p 显示建立相关链接的程序名

  • 然后就是kill -9 PID

https://www.runoob.com/w3cnote/linux-check-port-usage.html

# 杀死进程

kill -9 pid

# 后台运行脚本

  • ./xxx.sh & (终端退出时，就停了)
  • nohup ./xxx.sh & (不中断在后台运行。打印信息会输出到当前目录下的nohup.out中)
  • jobs查看当前shell环境中启动的任务情况。退出当前终端，再重新打开时，jobs看不到正在运行的脚本，ps -ef可以看到。
  • ps -ef | grep xxx.sh 查看正在运行脚本进程
  • ctrl + c是中断脚本。ctrl + z是切换到后台并暂停。
  • bg number 使其在后台执行，fg number 切换到前台执行。（number是使用jobs命令查询到的数字，不是pid）。

# ps -ef

ps: process status

-e: all the processes with in the system, not just the ones from the current terminal

-f: see a more detailed output

https://www.baeldung.com/linux/ps-command#1-listing-all-processes

ps aux 和ps -ef两者的输出结果差别不大，但展示风格不同。aux是BSD风格，-ef是System V风格。这是次要的区别，一个影响使用的区别是aux会截断command列，而-ef不会。当结合grep时这种区别会影响到结果。

推荐使用ps -ef。

# ll /proc/pid 查看进程所在的目录

# 常用记录

  • ps -ef | grep xxx
  • source ~／.bash_pro
  • tail -f xxx文件
  • telnet host port
  • lsof -i tcp :8080 查看端口
  • netstat -tunlp | grep 8080 查看端口占用
  • curl -X POST -H 'Content-Type:application/json' -H 'charset=UTF-8' -d '{"xxx":"xxx"}' http://xxxxx 在服务器里通过curl请求接口，这在生产的服务器中很有用。
