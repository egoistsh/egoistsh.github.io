今日所学

yarn和npm的区别

关键：

  • yarn和npm的作用一样。
  • yarn的优点：速度快、安装版本统一、简洁的输出

命令对比：

npm yarn
npm init yarn init
npm install yarn 或 yarn install
npm install react –save yarn add react
npm uninstall react –save yarn remove react
npm install react –save-dev yarn add react –dev

参考文章：

  • https://www.jianshu.com/p/254794d5e741

oracle–nvl()

syntax：nvl(expr1,expr2)

mean: If expr1 is null,then nvl returns expr2.If expr1 is not null,then nvl returns exp1.

今日所思

  • 找资源的原则：找优质、有趣的、目前需要，最好还是系统化的。
  • 明确自己的方向、目标，再去找相关资源，找关注up主的相关文章来看，无头苍蝇般地关注一堆是没用的。
  • 如何提高自己的水平：总是做一些你做的出来，但是难度大到只有再难一点点你就做不出来的事情。临界突破，就是这么个道理。
  • 自己根本还不算是个程序员。只不过是在追求当一个程序员的小人儿。随着思考，越活越明白越透彻。
  • 当你得到过金钱带来的快感，你就会越发想要加快金钱生产的速度。这十分有道理，能解释很多行为现象，但往往是不好的，要警惕，要懂得知足，要懂得居安思危，要懂得急流勇退。
  • 我崇尚计划，但不愿被计划框住。追求一种自由的计划。即有计划，但他是灵活变化的，他是可间断的，甚至是可中断的，。他的核心是计划提供指导，具体执行舒心。
