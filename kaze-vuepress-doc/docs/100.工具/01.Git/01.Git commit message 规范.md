title: Git commit message 规范
date: 2023-03-20 19:39:51
permalink: /pages/git/
---

message type：

- feat：新功能（feature）
- fix：修补bug
- perf：提高代码性能的变更
- docs：文档（documentation）
- style： 格式（不影响代码运行的变动），如格式化代码、删除空行
- refactor：重构（即不是新增功能，也不是修改bug的代码变动）
- test：增加测试
- chore：其他类型，如构建过程、依赖管理、辅助工具的变动

<br/>

代码类：

- feat
- fix
- perf
- style
- refactor

<br/>

非代码类：

- test
- ci
- docs
- chore
